# Quick QR

PCで見ているページを、スマホで「その場で」すぐ開ける Chrome 拡張

## なぜ作ったか

### 一般ユーザー向け

PCで調べ物をしていて、「この続きはあとでスマホで見よう」と思ったページ、結局そのままになりがちです。URLをコピーしてメモしたり、自分に送ったりするのも地味に手間です。

Quick QR は、拡張アイコンをクリックするだけで現在のページのQRコードを表示します。スマホのカメラで読み取れば、その場で同じページを開けます。

「あとで」を「今すぐ」に変えるための、シンプルな拡張です。

### 開発者向け

開発中や検証中に「いま見ているページをスマホで開いて確認したい」と思う場面はよくあります。URLを共有して端末で開くまでの手順は、回数が増えるほど面倒になり、作業のテンポを崩しがちです。

Quick QR は、アクティブタブのURLを取得してQRコードをローカルで生成し、ポップアップに表示します。外部APIを使わず拡張内で完結するため、軽量かつ手軽に導入できます。スマホで読み取るだけで、即座に同じページを開いて確認できます。

PCからモバイルへの移動コストを最小化するためのツールです。

## できること

- ポップアップを開いた瞬間に、現在のページのQRコードを表示します
- ワンクリックでURLをクリップボードにコピーできます
- QRコードをPNG画像としてダウンロードできます
- QRサイズを小/中/大から選べます
- ダークモードに対応しています（システム設定に追従）

## インストール方法

1. このリポジトリをダウンロードします（Code → Download ZIP）
2. ZIPを展開します
3. Chromeで `chrome://extensions` を開きます
4. 右上の「デベロッパーモード」をONにします
5. 「パッケージ化されていない拡張機能を読み込む」をクリックします
6. 展開したフォルダを選択します

## 使い方

QRコード化したいページを開き、Chrome右上の Quick QR アイコンをクリックします。ポップアップにQRコードが表示されるので、スマホのカメラで読み取って開いてください。

### ボタン

| ボタン | 機能 |
| --- | --- |
| URLをコピー | URLをクリップボードにコピー |
| QRを画像として保存 | QRコードをPNGでダウンロード |

## 注意点

以下のページではQRコードを生成できません。

- `chrome://` で始まるページ（設定、拡張機能など）
- `chrome-extension://` で始まるページ
- `about:` ページ
- `file://`（ローカルファイル）

対象外のページでは「このページはQR生成できません」と表示されます。

## 技術仕様

- Manifest Version: V3
- 権限: `activeTab`, `clipboardWrite`（最小限）
- 外部通信: なし（すべてローカルで完結）
- QR生成: qrcode-generator ライブラリ（同梱）
